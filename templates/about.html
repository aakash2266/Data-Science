About This Project

This web application predicts the BatteryDrop_Percent for an e-bike ride using a Machine Learning model built in Python. It takes into account **distance**, **speed**, **temperature**, and **rider weight** to estimate how much battery your ride will consume.

How the Model Works
  • Inputs: RideDistance_km, AvgSpeed_kmh, Temperature_C, RiderWeight_kg.
  • Preprocessed using StandardScaler for better model stability.
  • Trained using Linear Regression on real e-bike data.
  • Saved as model.pkl and loaded into Flask for live predictions.
Tech Stack

Python Flask scikit-learn Bootstrap 5

Project Goal

This project demonstrates the full ML deployment pipeline: data preprocessing, model training, saving with pickle, Flask integration, and building a modern UI suitable for class presentations and real demos.
